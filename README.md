# my-web

一个用于 **城市生活成本与跳槽决策对比** 的网页工具。  
它不是只比较年包数字，而是把 **收入、支出、通勤、时间、健康、家庭负担和主观偏好** 放在一起做综合判断。

## 在线预览

[点击打开网站](https://my-web-pied-nine.vercel.app)

## 项目简介

这个项目适合用于对比两份 offer 在不同城市下的真实生活价值，帮助用户回答这些问题：

- 同样的薪资，在不同城市到底哪个更值钱
- 跳槽后月可支配收入是增加还是减少
- 通勤、工作时长和健康压力会不会吞掉涨薪收益
- 在什么条件下，这次跳槽才真正成立

## 主要功能

- Offer 基础信息录入
- 城市与生活方式参数对比
- 月度收支拆解
- 时间与健康压力分析
- 情景模拟与阈值分析
- 风险预警与最终决策建议
